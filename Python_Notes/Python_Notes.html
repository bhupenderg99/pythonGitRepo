Set

Set : unique values, mutable, can use math operations like

  1. union
  1. intersect (common between both)
  1. difference (minus operator)
  1. to find unique elements in the list, simply assign it to set.
  1. useful example - you have 2 lists. patients which contains all members and enrolled patients which contains only eligible members. if we want to find out only the members who are not enrolled, we can use set difference operator.

    p = set(patients)

    e = set(enrolled_patients)

    not_eligible = p.difference(e) # this is like p - e i.e. elements in p not in e.
